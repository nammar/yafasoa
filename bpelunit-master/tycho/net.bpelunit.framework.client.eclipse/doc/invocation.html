Launching Test Suites

The BPELUnit Eclipse Client runs Test Suite Specification Files (.bpts) files. To get started, create a .bpts file in one of your projects in the workspace. A .bpts file may then be run as follows:

First option: Right-click the .bpts file, select Run As > BPELUnit Test Suite. The test run will begin immediately.

Second option: Select Run > Run... in the main menu bar. Select the BPELUnit TestSuite category and click the New Element button. The following page will be shown:

You need to provide:

  • A name for the launch configuration
  • A project in which the .bpts file resides
  • The .bpts file itself.

Click Run to run the test suite.
