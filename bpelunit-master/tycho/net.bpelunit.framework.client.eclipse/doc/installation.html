Installation

The BPELUnit Eclipse Client Installation consists of downloading two ZIP files:

  • The BPELUnit Eclipse Client itself
  • (Optionally) the Oracle UnDeployer, if you wish to deploy to the Oracle BPEL Server

Installing the client

The BPELUnit Eclipse Client ZIP file contains a feature and plugins. To install it,

  • Shut down Eclipse
  • Extracted the contents of the file directly into the Eclipse directory (the one which contains the features and plugins directories)
  • Start Eclipse

Installing the Oracle UnDeployer

The Oracle UnDeployer can be downloaded as a separate ZIP file. Extract this ZIP file somewhere on your harddisk. It contains an INSTALL file which contains instructions on how to install the UnDeployer.
